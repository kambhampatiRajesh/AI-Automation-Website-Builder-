##  Tech Stack

* **Frontend:** Streamlit
* **AI Framework:** LangChain
* **LLM:** Google Gemini / OpenAI (configurable)
* **Styling:** HTML5, CSS3
* **Dev Tools:** Git, VS Code
* **Optional:** LangSmith (debugging & tracing)

---

##  How It Works

1. User enters personal details, skills, and projects
2. Streamlit sends structured input to LangChain
3. LLM generates optimized portfolio content
4. AI builds complete HTML + CSS layout
5. Website is rendered or exported instantly
